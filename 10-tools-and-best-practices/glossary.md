# Glossary — Python & Computer Science Terms

A quick reference for key terms used throughout this guide.

---

## A

**Algorithm** — A step-by-step set of instructions to solve a problem.

**API (Application Programming Interface)** — A way for two programs to communicate. Like a waiter taking your order to the kitchen.

**Argument** — The actual value passed to a function when calling it. (vs. Parameter, which is the placeholder)

**Attribute** — A piece of data that belongs to a class/object. Example: `student.name`

---

## B

**Binary** — The number system using only 0 and 1. The native language of computers.

**Boolean** — A data type with only two values: `True` or `False`.

**Bug** — An error in code that causes unexpected behavior.

---

## C

**Class** — A blueprint or template for creating objects in OOP.

**Comment** — A line in code that Python ignores, meant for human readers. Starts with `#`.

**Compile** — Translating code into machine instructions. Python doesn't fully compile — it interprets.

**Concatenation** — Joining two strings together: `"Hello" + " World"`.

**Condition** — An expression that evaluates to True or False. Used in if/while statements.

**Constructor** — The `__init__` method in a class. Runs when a new object is created.

---

## D

**Data Type** — The kind of data a variable holds: int, float, str, bool, list, dict, etc.

**DataFrame** — A 2D table structure in Pandas (like a spreadsheet).

**Debugging** — The process of finding and fixing errors in code.

**Default Parameter** — A parameter with a pre-assigned value, used if no argument is given.

**Dictionary** — A Python data structure of key-value pairs: `{"name": "Ali", "age": 22}`

---

## E

**Encapsulation** — Bundling data and methods inside a class; hiding internal details.

**Environment Variable** — A variable stored in the operating system, outside your code. Used for secrets like API keys.

**Exception** — An error that occurs while a program is running.

---

## F

**Feature (ML)** — An input variable used to make predictions. Example: age, score, size.

**Float** — A decimal number data type: `3.14`, `-0.5`.

**For Loop** — Repeats code for each item in a sequence.

**Function** — A reusable, named block of code that performs a specific task.

**f-string** — A string with embedded expressions: `f"Hello, {name}!"`

---

## G

**Git** — A version control system that tracks changes in code over time.

**GitHub** — A cloud platform for hosting Git repositories and collaborating.

---

## I

**IDE (Integrated Development Environment)** — A full-featured code editor with tools for writing, running, and debugging code. (e.g., PyCharm)

**Import** — Loading a module or library into your Python file: `import math`

**Indentation** — Spaces at the start of a line. In Python, indentation defines code blocks.

**Index** — The position of an item in a sequence. Python uses 0-based indexing.

**Inheritance** — When a class derives properties and methods from another class.

**Instance** — A specific object created from a class. Every object is an instance.

**Integer (int)** — A whole number data type: `0`, `42`, `-7`

**Interpreter** — A program that reads and executes code line by line. Python is interpreted.

**Iteration** — Repeating a process (one pass through a loop = one iteration).

---

## K

**Keyword** — A reserved word in Python with special meaning: `if`, `else`, `def`, `class`, `return`, etc.

---

## L

**Label (ML)** — The correct answer/target in supervised learning.

**Library** — A collection of pre-written Python code you can import and use.

**List** — An ordered, mutable collection of items: `[1, "hello", 3.14]`

**Loop** — Code that repeats a block of instructions.

---

## M

**Method** — A function that belongs to a class.

**Module** — A Python file containing functions/classes that can be imported.

**Mutable** — Can be changed after creation. Lists and dicts are mutable; strings and tuples are not.

---

## N

**NaN (Not a Number)** — A special float value representing missing or undefined data. Common in Pandas.

**None** — Python's representation of "nothing" or "no value."

---

## O

**Object** — An instance of a class. Contains data (attributes) and behavior (methods).

**Object-Oriented Programming (OOP)** — A programming paradigm based on organizing code into objects.

**Operator** — Symbols that perform operations: `+`, `-`, `*`, `/`, `==`, `!=`, etc.

**Overfitting (ML)** — When a model memorizes training data and fails on new data.

---

## P

**Package** — A collection of Python modules. Installed with `pip`.

**Parameter** — A variable in a function definition that receives argument values.

**pip** — Python's package installer: `pip install pandas`

**Polymorphism** — Same method name, different behavior in different classes.

**Print** — The `print()` function that displays output to the screen.

---

## R

**RAM (Random Access Memory)** — Fast, temporary memory where programs run.

**Recursion** — A function that calls itself. Useful for problems with self-similar structure.

**REPL** — Read-Eval-Print Loop. Python's interactive shell where you type code line by line.

**Return** — Sends a value back from a function to the caller.

---

## S

**Scope** — Where a variable can be accessed. Local scope = inside a function. Global scope = entire file.

**self** — The first parameter of class methods; refers to the current object instance.

**Sequence** — An ordered collection (string, list, tuple) that can be indexed and iterated.

**Set** — An unordered collection of unique items: `{1, 2, 3}`

**Slice** — Extracting a portion of a sequence: `my_list[1:4]`

**Stack Trace / Traceback** — The error message Python shows, with file and line information.

**String (str)** — A sequence of characters (text): `"Hello, World!"`

**Supervised Learning** — ML where models learn from labeled training data.

---

## T

**Terminal / Command Line** — A text-based interface where you type commands directly to the OS.

**Token (AI/ML)** — A unit of text (roughly a word or word-piece) used by language models. APIs charge per token.

**Tuple** — An ordered, immutable collection: `(1, 2, 3)`

**TypeError** — Error raised when an operation is applied to an incompatible type.

---

## V

**Variable** — A named container that stores a value.

**Virtual Environment** — An isolated Python environment for a project, preventing package conflicts.

---

## W

**While Loop** — Repeats code as long as a condition is True.

---

## Z

**Zero-indexed** — Python counts positions starting from 0, not 1. The first item in a list is at index `0`.
