# Recommended Resources

> These are the best resources for continuing your Python journey after this guide.

---

## 📚 Free Online Resources

### Python Fundamentals
- **[Official Python Docs](https://docs.python.org/3/)** — The authoritative reference. Read it when you're stuck.
- **[Real Python](https://realpython.com)** — Well-written tutorials with practical examples.
- **[Python.org Beginner's Guide](https://wiki.python.org/moin/BeginnersGuide)** — Official beginner resources.
- **[W3Schools Python](https://www.w3schools.com/python/)** — Quick, searchable reference.

### Practice & Challenges
- **[LeetCode](https://leetcode.com)** — Algorithm challenges (start with Easy level)
- **[HackerRank Python](https://www.hackerrank.com/domains/python)** — Structured Python challenges
- **[Exercism.io](https://exercism.io/tracks/python)** — Exercises with mentor feedback
- **[Codewars](https://www.codewars.com)** — Fun, gamified coding challenges

### Data Science & AI
- **[Kaggle](https://www.kaggle.com)** — Free datasets, courses, and competitions
- **[Google's ML Crash Course](https://developers.google.com/machine-learning/crash-course)** — Free, excellent intro
- **[fast.ai](https://www.fast.ai)** — Practical deep learning from top educators

## 🔄 What to Do Next (Your Action Plan)

1. **This week:** Choose your specialization (Web Dev, Data Science, or Automation)
2. **This month:** Complete one tutorial in your chosen area
3. **In 3 months:** Build 3 portfolio projects and push them to GitHub
4. **In 6 months:** Apply for jobs or start freelancing
